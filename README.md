# 🎯 Whack-A-Mole Game

A fun and interactive **Whack-A-Mole** browser game built using **HTML, CSS, and JavaScript**. Test your reflexes by clicking the mole as it randomly appears on the game board before time runs out!

## 🚀 Features

- 🎮 Interactive 3 × 3 game board
- 🐹 Random mole appearance
- ⏱️ 60-second countdown timer
- 🏆 Real-time score tracking
- ▶️ Start game functionality
- ⏸️ Pause and Resume game
- 🔄 Restart game anytime
- 🔊 Background music and sound effects
- 📱 Responsive and colorful user interface
- 💻 Built using only HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6)
- DOM Manipulation
- CSS Transitions

## 🎮 How to Play

1. Click the **Start** button.
2. Watch the mole randomly pop out of different holes.
3. Click the visible mole to earn **1 point**.
4. Continue scoring before the **60-second timer** reaches zero.
5. Use the **Pause** button to temporarily stop the game.
6. Click **Restart** to begin a new game at any time.
7. Try to achieve the highest score before the timer ends.
